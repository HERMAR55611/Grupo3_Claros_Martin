# 🚀 Proyecto SpaceX - Análisis de Lanzamientos

## 📌 Descripción
Este proyecto realiza la extracción, análisis y visualización de datos de lanzamientos de SpaceX.

## 🛠 Tecnologías
- Python
- Pandas
- Matplotlib
- Streamlit

## 📊 Funcionalidades
- Extracción de datos desde API
- Limpieza y transformación
- Análisis exploratorio (EDA)
- Modelo básico de predicción
- Dashboard interactivo con Streamlit
