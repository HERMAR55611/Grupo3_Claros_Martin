# Proyecto Final - Minería de Datos SpaceX 🚀

Este proyecto desarrolla diferentes modelos de Machine Learning utilizando información de lanzamientos de SpaceX obtenida desde la API pública de SpaceX.

## 📌 Objetivo

Analizar y predecir el comportamiento de los lanzamientos espaciales mediante técnicas de minería de datos y aprendizaje automático.

---

# 📂 Contenido del Proyecto

## 1️⃣ Árbol de Regresión
Notebook enfocado en modelos de regresión utilizando árboles de decisión para estimar variables numéricas relacionadas con los lanzamientos.

Carpeta:
- `1_arbol_regresion`

---

## 2️⃣ Regresión Logística Binaria
Implementación de un modelo de clasificación binaria para predecir el éxito o fracaso de los lanzamientos.

Carpeta:
- `2_regresion_logistica`

---

## 3️⃣ Árbol de Clasificación Binaria
Desarrollo de un árbol de decisión para clasificación binaria utilizando variables del dataset SpaceX.

Carpeta:
- `3_arbol_clasificacion`

---
