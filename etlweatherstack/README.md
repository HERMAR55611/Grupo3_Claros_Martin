# 🚀 Proyecto SpaceX - Análisis de Lanzamientos

Este proyecto realiza un análisis de lanzamientos de SpaceX utilizando Python, Jupyter Notebook y Streamlit.

---

## 📁 Estructura del Proyecto

```
etlweatherstack/
├── C2L8 - JupyterNotebookLinux/
│   ├── spacex_final.ipynb   # Notebook principal
│
├── data/
│   └── spacex.csv           # Dataset
│
├── notebooks/
│   └── spacex_analysis.ipynb  # Análisis exploratorio
│
├── scripts/
│   ├── app.py               # App Streamlit
│   └── extract_spacex.py    # Script de extracción
│
├── requirements.txt
└── README.md
```

---

## ⚙️ Instalación

```bash
pip install -r requirements.txt
```

---

## ▶️ Ejecución

### 📊 Notebook

Abrir:

```
C2L8 - JupyterNotebookLinux/spacex_final.ipynb
```

### 🌐 App Streamlit

```bash
streamlit run scripts/app.py
```

---

## 📌 Funcionalidades

* Limpieza y transformación de datos
* Modelo de Machine Learning (Logistic Regression)
* Evaluación del modelo (accuracy y matriz de confusión)
* Visualización de datos
* Dashboard interactivo con Streamlit
